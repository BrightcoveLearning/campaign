Using the Heads-Up Display Chrome Extension

In this topic you will learn how to install and use the Heads-Up Display (HUD) Chrome browser extension.

The Heads-Up Display (HUD) is a Chrome extension that provides the ability to view performance metrics of videos on a live page.

Installing the Chrome extension

To install the Heads-Up Display Chrome extension, follow these steps:

  1. Click the Brightcove logo in the Chrome header to open the extension.

Using the Heads-Up Display

The Heads-Up Display is useful for viewing video analytics when a video that was published using Brightcove Campaign is on the page. To use the HUD, follow these steps:

  1. Click the Brightcove logo in the Chrome header to open the extension.
  2. If you don't have an active session with the Brightcove Campaign application, you will be prompted to login. Enter your email address and password. Click Sign In.

To sign out of the HUD, click Sign Out at the top of the HUD. Clicking Go To Web will open the Brightcove Campaign application in a browser window.

Reviewing HUD analytics

To review analytics for a live video on a page, open the HUD.

The HUD will display basic information about the video(s) on the page:

  • Video name
  • Video ID
  • Video Content Type
  • Published date

If there are multiple videos on the page, Previous Video and Next Video buttons will appear.

The following analytics data for the past 30 days will be displayed:

  • Loads - The number of times the video was loaded and ready for playback
  • Starts - The number of times the video started playing
  • Avg Percent Viewed - Average Time Viewed / Video length
  • Avg Time Viewed - Total Time Viewed / Starts

Hovering over a data point on the graph will display the date and value.

To see all analytics for the video, click the See all analytics link.
