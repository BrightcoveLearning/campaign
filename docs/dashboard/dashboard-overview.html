Overview of the Brightcove Campaign Dashboard

In this topic you will learn about the various Brightcove Campaign Dashboard pages that may be displayed.

The Brightcove Campaign Dashboard page provides a high-level overview of how your videos are performing. The Dashboard page that is displayed will vary based upon the status of your account and how your account is configured. The sections below display sample dashboard layouts. Find the dashboard that matches what you see in your account and follow the link to learn more about using that dashboard.

Advanced Dashboard

The Advanced Dashboard provides an overview of the best and worst performing videos in the account. This dashboard provides a Video Benchmark Score (VBS) for each video. The VBS compares the performance of your videos to the performance of other, similar videos of the same type. The Advanced Dashboard is only displayed in accounts which have enabled the Video Content Type metadata field for videos. This setting can be enabled as part of the account settings.

To learn more about the Advanced Dashboard, see Reviewing the Advanced Dashboard.

Basic Dashboard

The Basic Dashboard will provide an overview of the best and worst performing videos in the account. Graphs display the number of video impressions, video starts, avg. percent viewed and avg. time viewed. This dashboard does not provide a Video Benchmark Score (VBS) for each video and is unable to compare the performance of your videos to the performance of other, similar videos of the same type. To enable the VBS and comparative analytics, enable the Video Content Type metadata field for videos. This setting can be enabled as part of the account settings.

To learn more about the Basic Dashboard, see Reviewing the Basic Dashboard.

Getting Started Dashboard

The Getting Started Dashboard will appear the first time you login to the Brightcove Campaign application and will help you get started.

To learn more about getting started with Brightcove Campaign, see Getting Started with Brightcove Campaign.
