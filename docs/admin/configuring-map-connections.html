Configuring Marketing Automation Platform Connections

In this topic you will learn how to configure connections to marketing automation platforms.

Brightcove Campaign™ can be configured to integrate with popular marketing automation platforms (MAP) such as HubSpot, Eloqua and Marketo. When MAP integration is configured, video engagement data for videos published using Brightcove Campaign is synchronized to the selected marketing automation platform.

Note: You must be an account administrator to access the administration menu.
Note: The connection process will open popup windows in your browser. You should disable any browser popup blockers before setting up the connection.

To configure integration with marketing automation platforms, follow these steps:

  1. Login to Brightcove Campaign.
  2. In the navigation header, click the gear icon () and then MAP Connections. A list of current connections (if any) will appear.
  3. Click Add New Connection.
  4. Select the type of connection. See the appropriate topic for connection to that platform.
    • Eloqua
    • Marketo (REST API or Munchkin)
    • HubSpot (REST API or Client-Side API)
